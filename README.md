# Lab 🧪

A personal playground for learning, experimenting, and keeping small exercises in one place.

This repository contains a collection of mini-projects, kata-style tasks, proofs of concept, and quick demos across different programming languages, frameworks, and tools. The goal is simple: learn by building, keep it organized, and make it easy to revisit later.

---

## What’s inside

* **Language practice** (syntax, idioms, standard library)
* **Algorithms & data structures** (katas, coding challenges)
* **Framework experiments** (small apps, API samples, UI demos)
* **Tooling & DevOps** (Docker, CI, scripts, cloud snippets)
* **Notes & references** (short writeups, gotchas, links)

## How to run

Each folder should include its own `README.md` with:

* prerequisites (SDK/runtime version)
* setup steps
* how to run tests
* common commands

Example:

```bash
cd languages/csharp/some-exercise
dotnet restore
dotnet test
```

## Conventions

* Keep examples **small and focused**
* Prefer **clear naming** over clever naming
* Add a short README for anything non-trivial
* Include tests when it makes sense
* Commit often with meaningful messages
